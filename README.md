# Gregory Secure Systems

## Descripción

**Gregory Secure Systems** es un proyecto desarrollado por iniciativa propia como parte de un portafolio profesional, orientado al diseño, implementación y documentación de una infraestructura tecnológica corporativa. El proyecto fue desarrollado aplicando buenas prácticas de administración de sistemas, infraestructura y seguridad informática, con el propósito de evidenciar competencias técnicas en entornos empresariales.

---

## Objetivos

- Diseñar e implementar una infraestructura tecnológica corporativa basada en buenas prácticas de administración de sistemas y seguridad informática.
- Integrar servicios de infraestructura, administración de identidades, almacenamiento, monitoreo y seguridad para fortalecer la gestión y protección del entorno corporativo.
- Implementar controles de seguridad para proteger los activos tecnológicos, centralizar la administración y garantizar el acceso seguro a los recursos de la organización.
- Documentar el proceso de implementación y las configuraciones realizadas como evidencia técnica del proyecto.

---

## Tecnologías y servicios implementados

### Infraestructura y virtualización
- Windows Server 2022
- Hyper-V

### Servicios de infraestructura
- Active Directory Domain Services (AD DS)
- Active Directory Certificate Services (AD CS)
- DNS
- DHCP
- Group Policy (GPO)
- File Server
- File Server Resource Manager (FSRM)
- Internet Information Services (IIS)
- Windows Server Update Services (WSUS)

### Seguridad informática
- OPNsense Firewall
- Suricata (IPS)
- WireGuard VPN
- Wazuh SIEM
- Kali Linux

## Evidencias de implementación

El proyecto incluye evidencias técnicas del proceso de implementación de la infraestructura, entre ellas:

- Configuración de Active Directory.
- Administración de usuarios y grupos.
- Implementación de políticas de grupo (GPO).
- Configuración de DNS y DHCP.
- Implementación de File Server y FSRM.
- Implementación de IIS.
- Administración centralizada de actualizaciones mediante WSUS.
- Configuración y administración del firewall OPNsense.
- Implementación de WireGuard VPN para acceso remoto seguro.
- Integración de Wazuh SIEM para el monitoreo y gestión de eventos de seguridad.
- Infraestructura virtual implementada sobre Hyper-V.
