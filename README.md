# Gregory Secure Systems
## Descripción
**Gregory Secure Systems** es un proyecto académico y de portafolio profesional orientado al diseño e implementación de una infraestructura corporativa segura.
La infraestructura fue desarrollada utilizando:
- Windows Server 2022
- Active Directory
- Wazuh
- OPNsense
- Hyper-V
 
El objetivo del proyecto es presentar una infraestructura corporativa funcional orientada a la administración, monitoreo y protección de entornos empresariales, aplicando buenas prácticas de infraestructura y seguridad informática.
