# Gregory Secure Systems
## Descripción
**Gregory Secure Systems** Gregory Secure Systems es un proyecto desarrollado por iniciativa propia como parte de un portafolio profesional, orientado al diseño, implementación y documentación de una infraestructura tecnológica corporativa. El proyecto fue desarrollado aplicando buenas prácticas de administración de sistemas, infraestructura y seguridad informática, con el propósito de evidenciar competencias técnicas en entornos empresariales.
La infraestructura fue desarrollada utilizando:
- Windows Server 2022
- Active Directory
- Wazuh
- OPNsense
- Hyper-V
 
El objetivo del proyecto es presentar una infraestructura corporativa funcional orientada a la administración, monitoreo y protección de entornos empresariales, aplicando buenas prácticas de infraestructura y seguridad informática.
